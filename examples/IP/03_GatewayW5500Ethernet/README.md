# 03_GatewayW5500Ethernet

Planned gateway-side `EspNowIP` example using W5500 Ethernet as the uplink.

## What It Shows

- A planned Ethernet uplink for the gateway
- Reusing the same `routing + IPv4 NAT` gateway model with wired uplink
- A production-oriented alternative to Wi-Fi STA uplink

## Intended Use

Use this as a more deployment-friendly gateway example when a wired Ethernet uplink is available.
