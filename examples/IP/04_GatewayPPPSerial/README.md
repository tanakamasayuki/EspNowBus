# 04_GatewayPPPSerial

Planned gateway-side `EspNowIP` example using physical UART PPP as the uplink.

## What It Shows

- A planned PPP-over-UART uplink for the gateway
- A host PC connected over USB serial acting as the PPP peer
- Reusing the same gateway-side NAT model as other uplink types

## Intended Use

Use this as an alternative gateway example when Wi-Fi uplink is unavailable or intentionally avoided.
