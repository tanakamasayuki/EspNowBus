# 02_JoinAndUnicast

JOIN によるピア発見と、論理 ACK 付きユニキャスト送信を確認するサンプルです。

## このサンプルで確認できること

- JOIN トラフィックによる自動ピア発見
- `peerCount()` と `getPeer()`
- `bus.sendTo(...)` によるユニキャスト送信
- 受信側でのペイロード表示

## 使い方

1. 同じスケッチを 2 台以上のボードに書き込みます。
2. シリアルモニタを開きます。
3. ピアが見つかるまで待ちます。
4. `hello peer` が既知ピアのどれか 1 台に送られることを確認します。

最も基本的なピアツーピアのユニキャスト例として使えます。
