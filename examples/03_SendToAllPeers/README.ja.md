# 03_SendToAllPeers

既知の全ピアに対して、同じメッセージをユニキャストで送るサンプルです。

## このサンプルで確認できること

- `sendToAllPeers(...)`
- 単純なブロードキャストではなく、各ピアへのユニキャスト送信
- `peerCount()` / `getPeer()` によるピア一覧確認
- グループ全体への配信結果

## 使い方

1. 同じスケッチを 2 台以上のボードに書き込みます。
2. シリアルモニタを開きます。
3. ピアが見えるようになるまで待ちます。
4. 他ノードで `hello all peers` が受信されることを確認します。

ブロードキャストより重い代わりに、ピアごとのユニキャスト経路を確認できます。
