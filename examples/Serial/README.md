# Serial Examples

Examples under `examples/Serial` demonstrate the `EspNowSerial` and `EspNowSerialPort` layer built on top of `EspNowBus`.

## Included Examples

- `01_BasicPair`
- `02_ControllerBridge`
- `03_SilentDevice`
- `04_MultiSessionMonitor`
- `05_ESP32SerialCtlDevice`

These examples are intended to be mixed depending on the role of each board.
