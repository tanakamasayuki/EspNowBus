# 09_PhyRateOverride

Example that overrides the ESP-NOW PHY rate.

## What It Shows

- `cfg.phyRate`
- A lower PHY rate such as `WIFI_PHY_RATE_1M_L`
- Broadcast behavior under an explicit PHY setting

## How to Use

Flash the same sketch to multiple boards and use the serial log to confirm the requested PHY rate.
